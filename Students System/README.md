# 2.关键知识点
- 重点在于学号的自动生成，方法是设置一个为CStudent类所拥有的静态成员参数mCount，在构造函数中即可在设置学号时调用CStudent.mCount,并在构造完成时CStudent.mCount++,即可实现有序递增学号。
